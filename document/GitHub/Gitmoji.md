## gitmoji的使用

- [gitmoji项目地址](https://github.com/carloscuesta/gitmoji/)

?> gitmoji 的使用非常简单~  GitHub 也有详细的过程



### Using  gitmoji-cli

从命令行使用gitmoji，只需要用 NPM 安装 即可。gitmoji交互式客户端，用于在提交信息时使用表情符号。

```bash
npm i -g gitmoji-cli
```

在git 提交信息时，直接使用即可

```bash
git commit -m ':memo: 文档说明'
```
